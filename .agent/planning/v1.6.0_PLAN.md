## üì¶ Features for v1.6.0

### 1. **PyMongo Plugin** (Priority: P0)

**Why:** Complete MongoDB coverage (we have Motor async, add PyMongo sync)

**User Impact:** High - 300K+ developers use sync MongoDB  
**Effort:** 1 week  
**Difficulty:** Low (similar to Motor plugin)

**Implementation:**
```python
from pymongo import MongoClient
from timetracer.plugins import enable_pymongo

enable_pymongo()

client = MongoClient('mongodb://localhost:27017')
db = client.mydb

# All operations automatically captured
user = db.users.find_one({"email": "user@example.com"})
db.orders.insert_one({"user_id": user["_id"], "total": 99.99})
```
